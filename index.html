Generating SVG Files Programmatically with D3

The goal was to generate accessible graphs (SVG) for web content by using ARIA attributes. The SVG files were generated programmatically by executing a command in the command line. The D3v5 code was written in Typescript, NPM was used as package manager, gulp as task runner, and a CSV was taken as input. 3 charts were supported: Line-, Bar-, and Pie-Chart. E.g. gulp command gulp compile --file <ts-file> --dataset <csv-file>
